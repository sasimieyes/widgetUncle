# 블루투스 배터리 위젯

블루투스 장치별 배터리 잔량을 **작업표시줄에 직접 표시**하는 Windows 위젯입니다.
추가 프로그램 설치 없이 Windows 내장 PowerShell만으로 동작합니다.

두 가지 버전이 있습니다:

| 버전 | 실행 파일 | 표시 위치 |
|---|---|---|
| **바 버전 (권장)** | `start_bar.vbs` | 작업표시줄 오른쪽(알림 영역 옆)에 한 줄 바 |
| 트레이 버전 | `start_widget.vbs` | 알림 영역(트레이)에 장치별 아이콘 |

## 바 버전 (작업표시줄 직접 표시)

알림 영역(시계) 바로 왼쪽에 우측 정렬로 표시됩니다.

```
숫자 모드   : AK47-3 8%   Keyboard 100%   R1SE+ 55%   Buds3 85%
아이콘 모드 : AK47-3 [▮  ]  Keyboard [▮▮▮▮]  R1SE+ [▮▮ ]  Buds3 [▮▮▮ ]
숫자 내장  : AK47-3 [8]    Keyboard [100]    R1SE+ [55]    Buds3 [85]
```

### 기능
- **배경 투명(위장)** — 작업표시줄의 실제 색을 샘플링해 배경을 같은 색으로 칠하므로
  시각적으로 투명하게 보이면서, **바 사각형 전체에서 드래그·우클릭·더블클릭이 동작**
  (색상키 투명 방식은 글자 획 픽셀만 클릭이 잡혀 폐기)
- 장치별 `표시 이름 + 잔량` 표시 (숫자, 배터리 아이콘, 숫자 내장 배터리 아이콘 중 선택)
- 배터리 아이콘은 실제 잔량만큼 채워짐
- 아이콘 색상 모드:
  - **컬러** (기본): 정상 색상, **20% 이하 경고색, 10% 이하 위험색** 변경 가능
  - **그레이스케일**: 정상=#969696, 20% 이하=#4B4B4B, 10% 이하=#000000 고정
- 숫자 내장 배터리 아이콘은 글씨 색상과 게이지 경계 기준 색상 반전을 설정할 수 있음
- **현재 연결된 장치만 표시** (기본 켜짐) — 페어링만 되어 있고 꺼져 있는 장치는
  자동으로 숨겨지고, 켜서 연결되면 다음 갱신 때 자동으로 나타남
- **설정 창** (우클릭 → 설정..., Solarized Light + 부트스트랩 스타일):
  - 표시 방식 선택: 숫자 / 배터리 아이콘 / 배터리 아이콘 안에 숫자
  - 색상 모드 선택: 컬러 / 그레이스케일, 기본 색상·경고 색상·위험 색상·아이콘 숫자 색상 변경
  - 그레이스케일 선택 시 색상 선택 옵션 비활성화
  - 현재 연결된 장치만 표시 on/off
  - 장치 표시 설정 표: 표시 on/off + **표시 이름(별칭) 편집** (비우면 자동 약칭)
- 드래그로 가로 위치 이동 (자동 저장), 우클릭 → 위치 초기화
- 마우스 오버: 전체 장치명 + 상세 잔량 (이어버드 좌/우 `85% / 87%`)
- 더블클릭: 즉시 새로고침 / 우클릭 메뉴: 새로고침 주기(30초~5분), 자동 시작, 종료
- 진짜 전체화면 앱(게임·동영상)에서만 자동 숨김
  (시작 메뉴·검색·투명 오버레이 유틸리티는 오탐하지 않음)
- 작업표시줄 클릭 등으로 가려져도 1초 내 자동 복구
- 클릭해도 다른 앱의 포커스를 뺏지 않음 (WS_EX_NOACTIVATE)

### 실행 / 종료
- 실행: `start_bar.vbs` 더블클릭 (창 없이 백그라운드 실행)
- 종료: 바 우클릭 → 종료

## 트레이 버전

장치마다 트레이 아이콘 1개(배터리 % 숫자 + 게이지 바). 15% 이하 Windows 알림.

## 공통

### 자동 시작 등록
우클릭 메뉴 → **Windows 시작 시 자동 실행** 체크
(시작 프로그램 폴더에 바로가기 생성/삭제)

### 테스트 모드
```powershell
powershell -STA -File D:\project\widget\BtBatteryBar.ps1 -Test          # 조회+아이콘 렌더 테스트
powershell -STA -File D:\project\widget\BtBatteryBar.ps1 -SettingsTest  # 설정창 단독 테스트
```

### 동작 원리
PnP 장치 속성 `DEVPKEY_Bluetooth_Battery`(`{104EA319-6EE2-4701-BD47-8DDBF425BBE5} 2`)와
연결 상태 `System.Devices.Aep.IsConnected`(`{83DA6326-97A6-4088-9453-A1923F573B29} 15`)를
CIM(`Win32_PnPEntity.GetDeviceProperties`) 한 번의 호출로 함께 조회합니다.
배터리 정보를 OS에 보고하는 장치만 표시되며, 같은 장치의 인스턴스가 여러 개일 때는
연결된 인스턴스의 배터리 값을 우선 사용합니다 (이전 연결의 잔존 값 배제).

> 기술 메모 1: Windows 11 24H2/25H2의 XAML 작업표시줄은 `SetParent`를 통한 자식 창
> 부착을 즉시 되돌리므로, 바 버전은 최상위(TopMost) 창을 작업표시줄 영역 좌표에
> 정확히 겹치는 오버레이 방식을 사용합니다.
>
> 기술 메모 2: 전체화면 감지는 클릭 통과형 투명 오버레이(WS_EX_TRANSPARENT)와
> 셸 프로세스(시작 메뉴/검색 등)를 제외해 오탐을 막습니다.

## 설정 파일 (settings_bar.json)

| 항목 | 기본값 | 설명 |
|---|---|---|
| RefreshSec | 60 | 새로고침 주기(초) |
| DisplayMode | icon | `number`(숫자), `icon`(배터리 아이콘), `iconNumber`(아이콘 안 숫자) |
| IconColorMode | color | `color`(컬러) 또는 `gray`(그레이스케일) |
| NormalColor | #4CAF50 | 컬러 모드의 정상 잔량 색 (hex) |
| WarnColor | #FFCD3C | 컬러 모드의 20% 이하 경고 색 (hex) |
| CritColor | #FF5252 | 컬러 모드의 10% 이하 위험 색 (hex) |
| IconTextColor | #FFFFFF | 숫자 내장 배터리 아이콘의 게이지 위 글씨 색 (hex) |
| InvertIconText | true | 게이지가 채워진 영역과 빈 영역에서 숫자 색을 나눠 그림 |
| ConnectedOnly | true | 현재 연결된 장치만 표시 |
| HiddenDevices | [] | 숨길 장치 이름 목록 |
| Aliases | {} | 장치별 표시 이름 (원본 이름 → 별칭) |
| BarOffsetRight | 0 | 우측 기준점에서 왼쪽으로 띄울 거리(px) |

## 파일 구성

| 파일 | 역할 |
|---|---|
| `BtBatteryBar.ps1` | 바 버전 본체 (작업표시줄 오버레이) |
| `start_bar.vbs` | 바 버전 런처 (무창 실행) |
| `BtBatteryWidget.ps1` | 트레이 버전 본체 |
| `start_widget.vbs` | 트레이 버전 런처 |
| `settings_bar.json` / `settings.json` | 설정 (자동 생성) |
| `widget.log` | 오류 기록 |
